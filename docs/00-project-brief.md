# AI Project Radar｜AI 学习项目雷达

## 1. 项目简介

AI Project Radar 是一个帮助用户整理学习记录、AI 使用记录、教程笔记和项目想法的 AI 全栈 Web 应用。

用户可以输入一段学习记录，系统会调用 AI 进行结构化分析，生成总结、技能标签、问题发现、下一步行动，并进一步转化为项目卡片和简易 PRD，最后支持复制为 Markdown，方便保存到 Obsidian 等知识库中。

## 2. 目标用户

第一版主要面向：

- 正在学习 AI / 编程 / 全栈开发的学生
- 想把日常学习记录沉淀成项目成果的人
- 想用 AI 辅助整理想法、生成项目方案的人

## 3. 核心问题

用户在学习过程中会产生很多碎片内容，例如：

- 看教程后的收获
- AI 工具使用经验
- 项目灵感
- 每日学习复盘
- 技术问题和解决过程

但这些内容往往分散、杂乱，难以沉淀为可以展示的项目成果。

## 4. 解决方案

AI Project Radar 通过以下流程解决问题：

输入记录 → AI 分析 → 生成总结 → 生成项目卡片 → 生成 PRD → 复制 Markdown

## 5. 第一版核心闭环

第一版只完成一个最小可用闭环：

用户登录 → 新建记录 → AI 分析 → 保存分析结果 → 生成项目卡片 → 生成简易 PRD → 复制 Markdown

## 6. 第一版不做什么

第一版暂时不做：

- 全网爬虫
- 多人协作
- 付费系统
- 移动端 App
- 浏览器插件
- 向量数据库
- 复杂后台管理系统
- 自动批量导入 Obsidian
